El Sitio Web Más Alto del Mundo

Bienvenido a la página web más larga del mundo! Actualmente tiene 18,94 kilómetros de largo y es un verdadero placer hacer scroll. Disfrútalo! Haz scroll de forma manual o utiliza el ascensor, el cual ofrece teletransportación gratuita por el hipertexto a la velocidad de la luz.

|_

Increíble, verdad? Otra vez? Haz scroll de nuevo hasta llegar al principio, nosotros te esperamos aquí. Usa el ascensor o utiliza la combinación de teclas Ctrl + Inicio. El resto puede leer la letra pequeña. Esta página web es un experimento con CSS. Debido a su naturaleza tecnológica hay ciertas restricciones, pero cualquiera que tenga experiencia con éstas lo comprenderá. Esta página utiliza los siguientes elementos: un contenedor HTML; CSS (hojas de estilo en cascada); ningún otro truco. Una vez sabido esto, aquí estan los problemas conocidos e interesantes: Navegadores basados en un Gecko-Engine, como por ejemplo Firefox, muestran un comportamiento interesante si intentas hacer la página más larga de 18,939583 kilómetros. En este caso el contenedor principal encoje o incluso se colapsa. Internet Explorer ignora las referencias internas y en este caso no está claro si el contenedor realmente mide 18,94 kilómetros de largo. Si tienes una solución a este problema o incluso puedes aportar un elemento más largo, cuya representación esté soportada por más de un navegador, manda un correo electrónico a info@worlds-highest-website.com.
